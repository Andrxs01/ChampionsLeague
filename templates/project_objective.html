🎯 Objetivo Principal del Proyecto 🎯

El propósito fundamental de este proyecto es construir una aplicación web completa y robusta que sirva como una plataforma interactiva para la gestión y visualización de datos de la Liga de Campeones de la UEFA 2017/18. A través de este desarrollo, buscamos demostrar la capacidad de:

  • Dominio de Tecnologías Modernas: Implementar un backend sólido utilizando el framework web FastAPI de Python para construir una API RESTful eficiente y escalable.
  • Persistencia de Datos Fiable: Utilizar SQLAlchemy como un ORM (Object-Relational Mapper) para interactuar de manera fluida con una base de datos PostgreSQL, garantizando la integridad y persistencia de los datos.
  • Interfaz de Usuario Intuitiva y Atractiva: Diseñar y desarrollar un frontend con HTML5 y CSS3, empleando Jinja2 para la renderización de plantillas dinámicas. El objetivo es crear una experiencia de usuario (UX) clara, con una navegación coherente y un estilo visual que evoque la temática deportiva de la Champions League.
  • Funcionalidades CRUD Completas: Proporcionar todas las operaciones básicas de gestión de datos (Crear, Leer, Actualizar, Eliminar Lógicamente) para las entidades de Jugadores y Equipos.
  • Manejo de Contenido Multimedia: Integrar la capacidad de asociar y mostrar imágenes para cada jugador y equipo, enriqueciendo la información visual.
  • Búsqueda y Consulta Eficiente: Implementar funcionalidades de búsqueda que permitan a los usuarios encontrar rápidamente la información deseada.
  • Documentación Integrada: Ofrecer secciones de documentación dentro de la propia aplicación web que detallen las fases de planeación, diseño, los objetivos del proyecto y la información del desarrollador.

En resumen, este proyecto es una demostración práctica de la construcción de una aplicación web completa, desde la base de datos hasta la interfaz de usuario, con un enfoque en la calidad del código, la usabilidad y la estética.
